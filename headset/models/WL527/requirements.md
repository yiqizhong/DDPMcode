## Function descriptions
### Audio setting: 
	1: Noise control. there are three modes, which are ANC, Transparency, and Off. user can select one of those.  when user select ANC, it has Adaptive ANC, user can turn it on or off. Adaptive ANC should have a info tooltip to show "Automatically adjusts Active Noise Cancellation levels based on your surroundings"   when user select Transparency, user can adjust transparency strength from low to max, it has 3 stops.  
	2: Collaboration. it has Mic Noise Cancellation, user can turn it on and off. it has sidetone, user can turn it on and off. user can adjust sidetone when it's on, the range is 1 to 3.  collaboration,  Mic Noise Cancellation, and sidetone have the info tooltip. 
	3. Multimedia. there is a preset: Bass Boost, Speech Boost, and Custom. when custom is selected, it has EQ for user to adjust. Bass Boost is selected by default 


### Automated actions:
	
	1.Wear Detection. user can turn Wear Detection on or off.  under Wear Detection, it has Sensor Sensitivity, two modes: Low and Normal.   and there is another function, only for when headset is removed, it has Pause Music and Mute Microphone.  user can also turn on or off for both of these.  Wear Detection is the parent function.    info tooltip for: Wear Detection, Pause Music, Mute Microphone

	1.Auto OFF, user can turn it on or off. and user can adjust the Power off after inactive time, from 15 minutes, 30 minutes, 1 hour, 2 hours, 3 hours. to 5 hours.
